# **App Name**: Ram Dattu Kadiyala Portfolio

## Core Features:

- Hero Section: Displays name, title, tagline, and calls to action. Includes a dynamic background.
- About Section: Provides a professional summary and a headshot to introduce Ram Dattu Kadiyala.
- Skills Section: Categorized cards displaying core marketing and sales skills, tools, platforms, technical, and creative skills.
- Experience Section: Details previous experience with a focus on key achievements.
- Education Section: Highlights educational background and areas of focus.
- Contact Section: Provides contact information and a link to LinkedIn, with an optional contact form.

## Style Guidelines:

- Primary color: Royal Blue (#4169E1) to convey ambition and professionalism.
- Background color: Light desaturated blue (#E6EBF5), a lighter tint of royal blue to provide a modern look.
- Accent color: Silver (#C0C0C0) for highlights and interactive elements.
- Font: 'Poppins', a clean sans-serif, providing a modern marketing feel. Note: currently only Google Fonts are supported.
- One-page scroll layout with floating navigation bar for easy access to all sections.
- Smooth scroll animations for a seamless user experience.
- Subtle hover effects on interactive elements.
